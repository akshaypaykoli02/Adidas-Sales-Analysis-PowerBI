# Adidas Sales Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard built using Adidas US Sales data. The dashboard provides insights into sales performance, profitability, retailer contribution, product performance, and regional sales distribution.

## Tools Used

* Power BI Desktop
* Microsoft Excel

## Key Metrics

* Total Sales
* Operating Profit
* Units Sold
* Operating Margin
* Price per Unit

## Dashboard Features

* Monthly Sales Trend Analysis
* State-wise Sales Distribution Map
* Regional Sales Analysis
* Retailer Performance Analysis
* Product Category Analysis
* Interactive Filters and Slicers

## Files Included

* Adidas Sales Analysis.pbix
* Adidas US Sales Dataset.xlsx
* Project_Report.pdf

## Business Objective

To analyze Adidas sales data and generate actionable business insights using data visualization and business intelligence techniques.
